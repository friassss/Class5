El uso del Cajero ATM inicia con el programa pidiendo el PIN al usuario; si este se equivoca más de 3 veces se le retiene la tarjeta por un minuto y luego vuelve a iniciar. Una vez que el PIN correcto es ingresado, se le dan 5 opciones al usuario para usar según sus necesidades y se guarda un historial de sus acciones que este mismo también puede consultar. Cuando haya terminado, puede salir y el programa termina
